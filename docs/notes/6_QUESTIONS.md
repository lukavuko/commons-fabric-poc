# Development and Feature Questions

1. When users create an account, should their full name be required?
    - What are the implications of a fully anonymous web-app?
    - If a full name is required, what obligations do we have as a data host?
    - Can users choose whether to display their initial/first name/username?

2. What are the user facing data objects we want to present and store?
    - calendar events
    - community announcements
    - event invitations?

3. What are the different Event types we can include for communities?

4. Does each community need a human representative / point of contact for initial registration?
    - Does each community need an email registered under the organizations domain name?
    - What if there is no organization behind a community? What kind of group does this become?
